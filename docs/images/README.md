# Screenshots Directory

This directory contains screenshots of the OneCard Bot application.

## Image Files

Place the following images in this directory:

1. **login-screen.png** - Login interface showing the OneCard AI login form
2. **chat-empty-state.png** - Main chat interface with empty state and suggested actions
3. **chat-active.png** - Active chat conversation showing AI assistance in action

## Image Descriptions

### login-screen.png
Shows the login screen with:
- OneCard AI branding and logo
- User ID input field (pre-filled with example: `cust_1015a13f`)
- Password input field
- "Access Dashboard" button with gradient styling

### chat-empty-state.png
Shows the main application interface with:
- Left sidebar with navigation (AI Chat Helper, Templates, Projects, etc.)
- Central chat area with "How can I help you?" prompt
- Three suggested action cards: "Check Credit Limit", "Analyze Expenses", "Report an Issue"
- Right sidebar with history and recent chats
- Chat input field at the bottom

### chat-active.png
Shows an active conversation demonstrating:
- AI asking for customer ID
- User providing customer ID
- AI displaying transaction list with categories
- User requesting EMI conversion
- AI confirming the conversion with details
